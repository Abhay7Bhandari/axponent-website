# Contributing to Axponent Website

Thank you for your interest in contributing to the Axponent Website project! Here's how you can get involved:

## Fork the Repository
1. Visit the [Axponent Website GitHub page](https://github.com/Abhay7Bhandari/Axponent-website).
2. Click the ‘Fork’ button at the top right of the page.

## Clone the Forked Repository
Now that you've forked the repository, you need to clone it to your local machine:
1. Open your terminal.
2. Run the following command (replace `YOUR_USERNAME` with your GitHub username):
   ```bash
   git clone https://github.com/YOUR_USERNAME/Axponent-website.git
   ```

## Set Up the Environment
Ensure you have the necessary tools and dependencies installed. Refer to our project documentation for setup instructions specific to your development environment.

## Make Your Changes
1. Create a new branch for your changes:
   ```bash
   git checkout -b feature/your-feature-name
   ```
2. Make your changes and commit them:
   ```bash
   git add .
   git commit -m "Description of changes"
   ```

## Submit a Pull Request
1. Push your changes to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
2. Go to the original repository where you want to submit your changes.
3. Click on ‘Pull Requests’ and then ‘New Pull Request’.
4. Select your branch and follow the instructions to create the pull request.

Thank you for contributing! We appreciate your help in making the Axponent Website better!
